# Gettext

## First add this tap
`brew tap mongodb-ansible-roles/homebrew-gettext`

## Then install formula
`brew install gettext@0.20.2`

## Make sure symlink is correct
`brew link --overwrite gettext@0.20.2`

## Check version
`gettext --version`
